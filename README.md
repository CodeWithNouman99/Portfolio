# Nouman.dev 🌐

My personal developer portfolio built with React and Tailwind CSS to showcase my projects, technical skills, and journey as a Software Engineering student. The portfolio serves as a central place where recruiters, developers, and potential collaborators can learn more about me and explore my work.

## 🎯 Purpose

The goal of this portfolio is to create a professional online presence and showcase my projects, skills, and learning journey in web development.

It highlights:

* Personal introduction
* Technical skills
* Featured projects
* Education
* Contact section
* AI-powered assistant for visitor interaction

## ✨ Highlights

### 👨‍💻 About Me Section

A brief introduction about who I am, my background, and my interests in software development.

### 🛠️ Skills Showcase

Displays the technologies and tools I work with, including modern web development technologies.

### 🚀 Projects Section

Showcases my featured projects with descriptions and direct links to live demos and source code.

### 🤖 AI Assistant

Integrated chatbot that provides predefined answers to common questions about me, my skills, and my projects.

### 📄 Resume Access

Allows visitors to view or download my resume.

### 📬 Contact Section

Provides a way for visitors and recruiters to connect with me.

Optimized for desktop, tablet, and mobile devices.
